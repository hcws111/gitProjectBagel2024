## 使用簡介
1. 註冊或登入會員
2. 商品選購->點擊商品的購物車圖示(紅色加入、黑色尚未加入)
3. 點擊購物車->購物清單->設定商品數量
4. 輸入購買人資訊->提交
5. 會員專區->訂單資料->查看訂單狀態
